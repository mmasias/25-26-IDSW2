# Modelado de una Conversación

El reto 001 consta de un ejercicio únicamente de modelado haciendo énfasis en cómo se relacionan entidades entre sí.

He decidido de entre las opciones posibles modelar una **conversación**.

Esta desición se argumenta en que a primera vista tiene elementos claramente identificables, pero al mismo tiempo con un poco de análisis superficial se pueden hallar elementos implícitos que pueden ser interesantes de analizar, cosa que no se torne demasiado complejo, más sin embargo tampoco demasiado simple, un punto medio que me pareció adecuado.
