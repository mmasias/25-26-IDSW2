# Justificación de Relaciones Escogidas

## Relaciones Añadidas

Se introdujeron estas nuevas relaciones con el objeto de que existiese un *generador* de esos cuestionamientos, a partir de nuestros sentimientos. Esto porque los sentimientos estan presentes en general cada situación, percance o éxito en nuestra vida, lo que los hace muy influyentes a la hora de tomar decisiones.

Se introdujo un tipo de *ciclo vicioso* considerando que un sentimiento genera una pregunta con opciones las cuales tienen consecuencias probables y, con siendo consciente, si escojo una esa elección ejecutará una consecuencia real la cual usa emociones para construir sentimientos dentro de nosotros, que proceden a generar mas cuestionamientos.

### Sentimiento y Cuestionamiento

#### Agregación

La relación es de agregación porque el cuestionamiento, aunque nace del sentimiento, posee una lógica propia que puede persistir para el análisis incluso si el impulso emocional inicial varía. Eso hace que sean ambas independientes entre sí, y no solo los sentimientos afecten a la toma de la decisión.

### Sentimiento y Emoción

#### Composición

Se utiliza composición porque las emociones, en este contexto, son las piezas constituyentes del estado sentimental; si el sentimiento se disipa, la configuración específica de esas emociones como unidad funcional deja de existir.

### Consecuencia y Emoción

#### Uso

La consecuencia invoca una respuesta emocional para transformar un resultado objetivo en una valoración subjetiva. Este vínculo es el que permite que el sistema "aprenda", ya que la emoción resultante impactará finalmente en la capa de Sentimientos, cerrando el ciclo.
