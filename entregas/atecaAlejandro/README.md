# RETO-002
##  Principios Evaluados
Se han rastreado, justificado y corregido anti-patrones (mediante Pull Requests) en las siguientes categorías fundamentales:

* **Nombrado:** Eliminación de codificaciones redundantes y nombres desinformativos.
* **Comentarios:** Erradicación de código zombie y comentarios dogmáticos; fomento del código autoexplicativo.
* **Formato:** Mejora de la densidad vertical, afinidad y coherencia visual en estructuras de control.
* **Estándares:** Aplicación de encapsulación estricta, respeto a las convenciones de nomenclatura del lenguaje y erradicación de inconsistencias del idioma.
* **Consistencia:** Estandarización de abstracciones y comportamientos lógicos esperados (ej. orden de inserción en colecciones).
* **Código Muerto & Flujo:** Aplicación de *Early Returns* y eliminación de *Dead Stores*.
* **DRY:** Centralización de lógica duplicada mediante herencia y polimorfismo.
* **YAGNI:** Eliminación de diseño especulativo y métodos huérfanos.
